# Change Log

All notable changes to the "Auto Folder Colors" extension will be documented in this file.

## [1.0.0] - 2026-01-02

### Added
- Initial release
- Automatic workspace color switching based on active folder
- Status bar indicator showing current project
- Multi-project detection and tracking
- 8 predefined color options + custom color support
- Interactive color palette picker with emoji icons
- Commands for managing folder configurations:
  - Add Folder
  - Edit Folder Color
  - Remove Folder
  - List All Folders
- Real-time color updates when switching between files
- Support for multi-root workspaces
- Optimized performance with smart caching
- Professional code structure ready for publication

### Features
- Title bar, status bar, and activity bar coloring
- Active tab highlighting with project color
- Tab borders and breadcrumbs customization
- Tooltip showing multiple open projects
- Click status bar to quickly change colors

## Future Enhancements
- Workspace templates
- Import/export color configurations
- Theme presets (dark mode optimized, light mode optimized)
- Keyboard shortcuts for quick color switching
- Integration with workspace trust settings
